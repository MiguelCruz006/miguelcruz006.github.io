Mon profil technique

Passionné par les infrastructures et la sécurité informatique, je prépare mon BTS SIO option SISR en formation à distance au CNED. Ma formation couvre les réseaux Cisco, l'administration Windows Server et Linux, la virtualisation, et le cloud Microsoft Azure.

De la configuration d'équipements Cisco IOS (VLANs, HSRP, STP) à l'automatisation d'infrastructures cloud via Terraform sur Azure, j'ai développé une vision complète des environnements d'infrastructure modernes. En parallèle, je me forme activement en cybersécurité sur TryHackMe, Hack The Box et Root-Me et je prépare la certification EHC (Ethical Hacker Certified).

Infrastructure Windows Azure IaC
Déploiement d'une infrastructure cloud complète sur Microsoft Azure via Terraform : 2 VMs Windows Server 2022, 1 poste client Windows 11, un réseau virtuel segmenté (VNet 10.0.0.0/16) et un NSG avec règles RDP et SSH. L'ensemble des 15 ressources Azure est défini en code HCL versionné sur Git et déployable en une seule commande terraform apply.

Objectif professionnel

À l'issue de mon BTS SIO SISR, mon objectif est d'intégrer une formation d'ingénieur en cybersécurité. Je vise un rôle d'Ingénieur Cybersécurité — analyste SOC, pentest ou architecte sécurité — dans des environnements cloud et hybrides.
